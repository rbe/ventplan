WAC Readme
==========

Zur Installation kopieren Sie das Archiv "wac_NNN_OS.zip" von dieser CD
auf Ihre Festplatte und entpacken es. NNN steht dabei für die Version
(z.B. 6.0.11) und OS für das Betriebssystem (win = Windows, linux = Linux
und mac = Mac OS X). Beispiel für Windows: wac_6010_win.zip.

Diese Version ist auch von CD startfähig. Beachten Sie bitte, dass das Programm
dann langsamer läuft.

Start unter Windows
-------------------
Zum Starten der Applikation gehen Sie in das Verzeichnis "wac" und starten Sie
die Applikation über das Skript "wac.bat".

Start unter Linux, Mac OS X
---------------------------
Zum Starten der Applikation gehen Sie in das Verzeichnis "wac" und starten Sie
die Applikation über das Skript "wac". Öffnen Sie dazu ein Terminal und geben
Sie ein:
$ cd wac
$ ./wac
